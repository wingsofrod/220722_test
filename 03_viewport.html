데스크탑 퍼스트 : max
html의 기본 시스템 사이즈는 font-size:16px이다. 사이즈를 기정하지 않으면 기본적으로 시스템 자이즈가 적용된다. html{font-size:16px;} 에서 16px은 1rem, 17px은 1.063rem이 된다. 이러면 계산하기 어려우므로, html{font-size:62.5%;} 또는 html{font-size:10px;} 로 하면 10px은 1rem, 12px은 1.2rem, 14px은 1.4rem 이 되기 때문에 따로 계산할 필요가 없다.
  • default 10px / 10px = 1rem
  • default 10px / 12px = 1.2rem
  • default 10px / 14px = 1.4rem
  • default 10px / 16px = 1.6rem
  • default 10px / 18px = 1.8rem
  • default 10px / 20px = 20rem
  • default 10px / 22px = 2.2rem
  • default 10px / 24px = 2.4rem
  • 배경컬러 rgb(255,160,122)
  • default 16px 16px = 1rem(base)
  • 18px = 1.125rem
  • 20px = 1.25rem
  • 22px = 1.375rem
  • 24px = 1.5rem
  • 26px = 1.625rem
  • 28px = 1.75rem
  • 30px = 1.875rem
  • 32px = 2rem
  • http://pxtoem.com/
미디어쿼리로 css를 처리할 때 브레이크 포인트에서 html의 폰트 사이즈만 변경 해주면 하위 rem 단위의 크기가 전부 변경되어 편리하다. https://www.w3schools.com/cssref/css_pxtoemconversion.asp
